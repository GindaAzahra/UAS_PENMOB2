## 📱 Sistem Presensi Karyawan Berbasis Mobile

Aplikasi ini merupakan sistem presensi karyawan berbasis mobile yang dikembangkan untuk membantu proses pencatatan kehadiran karyawan secara digital, terstruktur, dan real-time.

Aplikasi memiliki dua peran pengguna, yaitu karyawan dan staf:

- Karyawan dapat melakukan presensi masuk dan pulang, melihat riwayat kehadiran, serta mengajukan izin atau sakit.

- Staf bertugas memantau data presensi karyawan, melihat riwayat kehadiran, serta memberikan persetujuan atau penolakan terhadap pengajuan izin dan sakit.

Sistem ini dirancang untuk meningkatkan efisiensi, akurasi, dan transparansi dalam pengelolaan kehadiran dibandingkan dengan sistem presensi manual.

## ✨ Fitur Utama

Presensi masuk dan pulang karyawan

Riwayat presensi harian

Pengajuan izin dan sakit

Persetujuan izin oleh staf

Dashboard monitoring kehadiran

## 🛠️ Teknologi yang Digunakan

Flutter (Mobile Application)

Firebase Authentication (Autentikasi pengguna)

REST API / Mock API (Simulasi backend)

Provider (State Management)
