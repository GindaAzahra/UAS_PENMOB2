## ğŸ“Œ Deskripsi Proyek
Aplikasi Food Ordering ini merupakan proyek UAS Pemrograman Mobile 2 (PEMOB 2) dengan konsep seperti ShopeeFood, yaitu aplikasi pemesanan makanan secara online. Pengguna dapat melakukan registrasi, login, melihat daftar makanan, memesan makanan, melihat pesanan, hingga mengelola profil pengguna.

Aplikasi ini dibangun menggunakan Flutter dan menerapkan konsep Hybrid Server, REST API, Firebase Authentication, serta deployment ke Web (PWA) dan Android (APK).

## ğŸ¯ Tujuan Proyek
* Memenuhi persyaratan tugas besar UAS PEMOB 2 dengan membangun aplikasi mobile dan web berbasis Flutter.

* Menggunakan Supabase sebagai:

  * Authentication

  * Database,

  * REST API

* Mengimplementasikan animasi dan transisi, seperti:

   * Animasi pada Splash Screen saat aplikasi pertama kali dibuka,

   * Animasi transisi halaman, termasuk efek slide pada halaman login,

* Melakukan deployment aplikasi ke Web dalam bentuk Progressive Web App (PWA) menggunakan layanan hosting Netlify, sehingga aplikasi dapat diakses melalui browser.
